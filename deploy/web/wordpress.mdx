Typebot possui um [plug-in nativo para WordPress](https://wordpress.org/plugins/typebot/) que ajuda a incorporar typebots em seu site WordPress.

É claro que, antes de usá-lo, você precisa criar e publicar seu primeiro typebot.

  <img src="/images/embeddings/wordpress-preview.png" alt="Visualização do plug-in WP" />
</Frame>

O trecho de código para colar é facilmente configurável na aba "Compartilhar" do seu bot depois de clicar no botão "WordPress".

## Páginas excluídas

A entrada de páginas excluídas é uma lista separada por vírgulas de páginas onde você não quer que seu typebot apareça.

Exemplos:

- `/app/*` excluirá todas as páginas começando com `/app/`
- `/app` excluirá apenas a página `/app`
- `/app?param=1` excluirá apenas a página `/app` **e** com o parâmetro de consulta `param` definido como `1`
- `/app?param=*` excluirá a página `/app` **e** com o parâmetro de consulta `param` definido como qualquer coisa
- `/app/*?param=*` excluirá todas as páginas começando com `/app/` **e** com o parâmetro de consulta `param` definido como qualquer coisa

## Personalize a experiência do usuário

Você pode aproveitar as [variáveis pré-preenchidas](../../editor/variables#prefilled-variables) e injetar suas informações de usuário diretamente em seu typebot para que a experiência seja totalmente personalizada para o seu usuário.

Aqui estão as variáveis disponíveis no WordPress, certifique-se de criá-las no menu suspenso de variáveis do seu typebot:

<Frame style={{ maxWidth: '400px' }}>

A única coisa que você precisa fazer para habilitar isso é:

- Usar o [plug-in do WordPress Typebot](https://wordpress.com/plugins/typebot)
- Ter as variáveis declaradas no seu Typebot com as sintaxes exatas. Por exemplo, para o e-mail, certifique-se de que sua variável seja escrita como `WP Email`. Estas não funcionarão: `wp Email`, `WP email`.

## Seu typebot não está aparecendo?

### Você tem litespeed com "Localise Resources" habilitado

Há uma caixa onde há uma lista de URLs que ele localiza, um deles era ‘https://cdn.jsdelivr.net’. Este URL deve ser removido dele, já que é usado para importar a biblioteca de incorporação.

## Você tem litespeed com cache habilitado

Certifique-se de inserir `web.js` e `typebot` na caixa de texto de JS Excludes e JS Deferred Excludes em Tuning Settings.

### Você tem um plugin de cache

Plugins como WP Rocket impedem que o Typebot funcione.

Para o WP Rocket:

1. Vá para Configurações > WP Rocket > Excluded Inline Javascript:

  <img src="/images/embeddings/wp-rocket.png" alt="WP Rocket" />

2. Digite "typebot"
3. Salve
