---
title: Trecho de Incorporação de Script
---

A opção de incorporação de script é útil apenas se você não tiver acesso à árvore HTML de sua aplicação ou se seu construtor de sites permitir apenas que você insira trechos de script em linha.

Caso contrário, é preferível seguir as instruções de incorporação [HTML & Javascript](./libraries/html-javascript) porque os trechos de script são apenas scripts que injetarão o código do método de incorporação HTML & Javascript.
