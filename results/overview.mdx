Uma vez que seu bot é [publicado](/editor/publish), você começa a coletar resultados dos seus usuários. Você pode ver todos os resultados na aba `Resultados` do seu bot.

<Note>
  Você pode se surpreender ao ver resultados parcialmente preenchidos. Isso é normal e esperado. Seu typebot coleta as respostas assim que são preenchidas pelo usuário, mesmo que o usuário não complete toda a conversa. Isso é útil para entender onde os usuários estão desistindo e para melhorar seu bot. Esta é uma das maiores características do Typebot.
</Note>

### Exportar todos os resultados para um arquivo CSV

Você pode exportar todos os resultados para um arquivo CSV. Isso irá baixar um arquivo CSV com todos os resultados do bot atual.

A opção `Incluir blocos excluídos`, se habilitada, incluirá respostas de blocos que não existem mais (faziam parte de uma versão anterior do seu bot).

### Reorganizar e ocultar colunas específicas
