A aba de tema permite que você personalize a aparência do seu typebot.

## Global

Esta seção permite que você habilite ou desabilite a marca do typebot, altere a fonte e o fundo do seu bot.

## Chat

Esta seção permite que você personalize todos os componentes do chat (avatares, balões, entradas, etc.).

Clique no avatar do bot para alterar a imagem:

## CSS Personalizado

Você também pode personalizar ainda mais adicionando qualquer CSS personalizado que desejar. Para isso, clique com o botão direito no bot na página de Tema e inspecione o elemento que deseja personalizar:

<Frame>
  <img src="/images/theme/custom-css.png" alt="Inspeção de CSS personalizado" />
</Frame>

Por exemplo, se eu quiser que meus botões sejam mais arredondados e tenham uma cor de gradiente elegante, posso adicionar isso ao CSS personalizado:

```css
.typebot-button {
  border-radius: 40px;
  background-image: linear-gradient(to right, #e052a0, #f15c41);
  border: none;
}
```

<Frame>

Personalizar a cor de um único botão

Graças ao CSS personalizado, você pode personalizar a cor de um único botão, por exemplo, usando o atributo data-itemid:

```css
[data-itemid='cl3iunm4p000f2e6gfr8cznnn'] {
  background-color: gray;
  border-color: gray;
}
```

Para encontrar o ID do item de um botão, clique com o botão direito no botão e inspecione o elemento:
