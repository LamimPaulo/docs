Com o bloco Zemantic AI, você pode pesquisar e recuperar resultados ou resumos de LLM dos seus documentos armazenados no Zemantic AI.

## Configurações

Esta integração requer uma conta Zemantic AI. Se você ainda não tem uma, pode criar uma [aqui](https://zemantic.ai/).

O bloco possui as seguintes configurações:

<Frame>
  <img
    src="/images/blocks/integrations/zemanticAi/settings.png"
    alt="Configurações Zemantic AI"
  />
</Frame>

- Conta Zemantic AI: criar ou selecionar as credenciais do Zemantic AI que você deseja usar.
- ID do Projeto: O ID do projeto contendo os documentos que você deseja pesquisar.
- Pergunta ou Consulta: A pergunta que você deseja fazer aos seus documentos do Zemantic AI.
- Resultados Máximos: O número máximo de resultados que você deseja recuperar.

### Configurações de Prompt

- Prompt do Sistema: O prompt que você deseja usar para orientar o comportamento do LLM.
- Prompt: O prompt que você deseja usar para resumir seus documentos.

## Solução de Problemas

### Mensagem de erro: "Zemantic AI block returned error"

Isso significa que seu bloco Zemantic AI não está configurado corretamente. Verifique o seguinte:

- Você selecionou uma conta Zemantic AI.
- Você preencheu o campo Pergunta ou Consulta.

### Retorna uma mensagem vazia

Ou você configurou o bloco incorretamente ou pode ter excedido os limites de contexto do LLM. Você pode tentar reduzir o número de resultados a serem recuperados ou encurtar seu prompt.
