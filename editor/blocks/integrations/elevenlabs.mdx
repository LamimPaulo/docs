Este bloco permite integrar a API da ElevenLabs em seu typebot.

## Converter texto em fala

Permite transformar um texto em fala usando vozes e modelos da ElevenLabs. Esta ação cria automaticamente um link temporário para o arquivo de áudio. Este link pode ser usado em uma [bolha de áudio](../bubbles/audio), por exemplo.
