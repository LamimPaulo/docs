---
title: Chatwoot
---

O bloco de integração do Chatwoot permite que você abra um widget do Chatwoot para permitir que seu usuário fale diretamente com um humano. Isso permite que você adicione uma camada de Chat ao vivo ao seu typebot.

## Requisitos

Para esta integração, você precisa de uma conta no Chatwoot e criar uma caixa de entrada "Website":

<Frame>
  <img
    src="/images/blocks/integrations/chatwoot/website-inbox.png"
    alt="Criar caixa de entrada do site"
  />
</Frame>

## Configuração

Insira um bloco Chatwoot onde você deseja acionar o widget:

<Frame>
  <img
    src="/images/blocks/integrations/chatwoot/options.png"
    alt="Bloco Chatwoot"
  />
</Frame>

Para encontrar seu token de site, vá para o Chatwoot nas configurações da sua caixa de entrada:

<Frame>
  <img
    src="/images/blocks/integrations/chatwoot/website-token.png"
    alt="Encontrar token do site"
  />
</Frame>

Você pode preencher informações do usuário adicionando variáveis coletadas às entradas de "Definir detalhes do usuário".

Por exemplo, se você definir a entrada "Email" como "john@gmail.com", o Chatwoot associará automaticamente este e-mail ao usuário atual.

## Atributos personalizados

Você pode adicionar esses atributos personalizados que o Typebot preencherá automaticamente para você:

### URL do resultado

Você pode vincular a URL do resultado atual à conversa do Chatwoot criando este atributo personalizado:

<Frame>
  <img
    src="/images/blocks/integrations/chatwoot/custom-attribute.png"
    alt="Atributo personalizado Chatwoot"
  />
</Frame>

### Definir comportamento do usuário

Se você estiver preenchendo informações do usuário, por padrão, se deixar a entrada de ID vazia, ele definirá o ID do usuário do Chatwoot como o `Email` ou o ID do Resultado.

Definir o `Email` como o ID nos permite evitar que contatos não sejam mesclados corretamente (https://github.com/chatwoot/chatwoot/issues/2811)
