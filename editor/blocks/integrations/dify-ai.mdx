Este bloco permite que você integre o assistente da Dify.AI em seu typebot.

## Criar Mensagem de Chat

Esta ação envia uma mensagem do usuário para seu agente. Em seguida, você pode salvar `Resposta` em uma variável e exibi-la em seu typebot.

Você deve fornecer os seguintes parâmetros:

- `Consulta`: A mensagem do usuário que você deseja enviar para seu agente.
- `ID da Conversa`: O ID da conversa que você deseja usar para esta mensagem. Se você não fornecer um, uma nova conversa será criada. O conteúdo desta variável será atualizado automaticamente se uma nova conversa for criada.
- `Usuário`: O e-mail do usuário usado para identificar o usuário na conversa.
