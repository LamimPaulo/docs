O bloco de entrada de Website permite que você peça ao seu usuário uma URL. Ele verificará se está devidamente formatada.

A mensagem de tentativa será exibida sempre que o Typebot detectar que o website não está devidamente formatado.

Ele não verificará se o endereço do website é **válido**. Para isso, você precisará acionar um [bloco de Webhook](/editor/blocks/integrations/webhook) e chamar uma API de serviço de validação de website.
