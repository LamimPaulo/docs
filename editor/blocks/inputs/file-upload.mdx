O espaço reservado aceita [HTML](https://en.wikipedia.org/wiki/HTML).

## Limite de tamanho

Há um limite fixo de 10MB por arquivo enviado. Se você quiser que seus respondentes enviem arquivos maiores, deve pedir que façam o upload de seus arquivos para um serviço de armazenamento em nuvem (por exemplo, Google Drive, Dropbox, etc.) e compartilhem o link com você.

## Visibilidade

Esta opção permite que você escolha entre gerar URLs públicas para os arquivos enviados ou mantê-los privados. Se você escolher manter os arquivos privados, poderá ver o arquivo apenas se estiver logado em sua conta Typebot.

Observe que, se você escolher manter os arquivos privados, não poderá usar a URL do arquivo com outros blocos, como Anexo no bloco Enviar e-mail ou outros. Esses serviços não poderão ler os arquivos.

Por padrão, esta opção está definida como `Auto`. Isso significa que os arquivos serão públicos se enviados pelo runtime web, mas privados se enviados pelo runtime WhatsApp.
