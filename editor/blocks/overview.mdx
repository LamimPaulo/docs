title: Visão Geral
---

Os blocos são os componentes básicos de uma conversa no Typebot. Você pode encadear qualquer bloco para criar fluxos de conversa complexos. Quando você arrasta e solta um bloco no fluxo do Typebot, por padrão ele será inserido em um grupo que pode ter um título. Os blocos são sempre contidos em grupos.

Existem várias categorias de blocos para facilitar a busca pelo bloco certo para suas necessidades:

## Balões

Os blocos de balões são usados para exibir balões que podem ser usados para mostrar informações ao usuário.

## Entradas

Os blocos de entrada são usados para solicitar entrada do usuário. Isso interromperá a conversa e solicitará ao usuário que forneça uma entrada.

## Lógica

Os blocos de lógica são usados para realizar operações em segundo plano. Eles não são visíveis para os usuários.
