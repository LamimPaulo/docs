---
title: Menu de Contexto
ícone: bars
---

Você pode clicar com o botão direito em diferentes elementos no construtor para acionar um menu com diferentes ações.

Se você clicar com o botão direito em um grupo, verá um menu com as seguintes opções:

<Frame>
  <img src="/images/group-context-menu.png" alt="Menu de contexto" />
</Frame>

- **Duplicar** para criar um grupo duplicado com os mesmos blocos e configurações
- **Excluir** para remover o grupo

Se você clicar com o botão direito em um bloco, verá um menu com as seguintes opções:

- **Duplicar** para criar um bloco duplicado abaixo do bloco duplicado com as mesmas configurações
- **Excluir** para remover o bloco

Se você clicar com o botão direito em uma aresta, verá um menu com as seguintes opções:

<Frame>
  <img src="/images/edge-context-menu.png" alt="Menu de contexto" />
</Frame>

- **Excluir** para remover a aresta
